# Anaconda 源使用帮助

## 地址

<https://mirrors.nju.edu.cn/anaconda/>

## 说明

Anaconda 仓库镜像

## 使用说明

Anaconda 安装包可以在 <https://mirrors.nju.edu.cn/anaconda/archive/>
下载。

添加NJU仓库镜像：

    conda config --add channels https://mirrors.nju.edu.cn/anaconda/pkgs/free/
    conda config --set show_channel_urls yes
